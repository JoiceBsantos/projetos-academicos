# 💊 Sistema de Banco de Dados - Rede de Farmácias

Projeto desenvolvido no 1º semestre da faculdade para a disciplina de Banco de Dados.

---

## 💡 Contexto do Projeto

A escolha do tema "Rede de Farmácias" foi baseada na minha experiência profissional como farmacêutica, atualmente em transição para a área de tecnologia.

O objetivo foi unir conhecimentos da área da saúde com conceitos de banco de dados, criando uma solução mais próxima de um cenário real, considerando rotinas como controle de estoque, vendas e gestão de clientes.

Essa abordagem permitiu desenvolver o projeto com uma visão mais prática e alinhada às necessidades do mercado.

---

## 📌 Descrição

Este projeto consiste na modelagem e implementação de um banco de dados para uma rede de farmácias, com o objetivo de organizar e gerenciar informações essenciais do sistema.

O banco permite controlar:

- Clientes  
- Produtos  
- Estoque  
- Funcionários  
- Pedidos  
- Fornecedores  
- Formas de pagamento  

---

## 🧠 Modelagem do Banco

O desenvolvimento foi dividido em três etapas principais:

### 🔹 Modelo Conceitual
Criação do diagrama Entidade-Relacionamento (ER), definindo entidades, atributos e relacionamentos.

### 🔹 Modelo Lógico
Transformação do modelo conceitual em tabelas estruturadas utilizando MySQL Workbench.

### 🔹 Modelo Físico
Implementação do banco de dados com SQL, incluindo:

- Criação de tabelas  
- Definição de chaves primárias e estrangeiras  
- Relacionamentos entre entidades  

---

## 🗂️ Estrutura do Projeto

- `script-banco-farmacia.sql` → Script completo do banco de dados  
- `projeto-banco-de-dados.pdf` → Documentação completa com diagramas e explicações  

---

## 🚀 Como Executar

1. Abra o MySQL Workbench (ou outro SGBD compatível)  
2. Execute o arquivo:

```sql
source script-banco-farmacia.sql;
```

3. O banco `rede_farmacias` será criado automaticamente
