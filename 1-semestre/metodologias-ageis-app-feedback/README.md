# 📱 Projeto de Metodologias Ágeis - App de Feedback para Restaurantes

Projeto desenvolvido no 1º semestre da faculdade para a disciplina de Metodologias Ágeis.

---

## 💡 Contexto do Projeto

Este projeto foi criado com o objetivo de aplicar conceitos de metodologias ágeis no desenvolvimento de uma solução digital.

A proposta foi desenvolver a ideia de um aplicativo voltado para coleta de feedback de clientes em restaurantes, permitindo melhorias contínuas no atendimento e nos serviços oferecidos.

---

## 📌 Descrição

O projeto consiste na prototipação de um aplicativo com foco na experiência do usuário, onde clientes podem avaliar e fornecer feedback sobre restaurantes.

A solução foi pensada para:

- Melhorar a comunicação com clientes  
- Coletar opiniões em tempo real  
- Auxiliar na tomada de decisão dos estabelecimentos  

---

## 🎯 Objetivo

Aplicar conceitos de metodologias ágeis como:

- Organização de ideias  
- Prototipação rápida  
- Foco no usuário  
- Desenvolvimento iterativo
