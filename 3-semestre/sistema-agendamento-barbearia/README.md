# 💈 Sistema de Agendamento para Barbearia

Projeto desenvolvido no 3º semestre do curso de Análise e Desenvolvimento de Sistemas como parte da disciplina de Programação Orientada a Objetos.

---

## 📌 Sobre o Projeto

O objetivo deste projeto foi desenvolver um sistema desktop para gerenciamento de agendamentos em uma barbearia, permitindo que usuários realizem cadastros, agendamentos e que administradores gerenciem os serviços disponíveis.

A aplicação foi desenvolvida em **Java**, utilizando **Java Swing (JFrame)** para a interface gráfica e **PostgreSQL** para o banco de dados.

---

## 🎯 Objetivo Acadêmico

O projeto teve como objetivo consolidar conhecimentos do semestre nas áreas de:

- Programação Orientada a Objetos (POO)

- Modelagem de Dados
- Integração com Banco de Dados
- Estruturas de Dados
- Desenvolvimento de interfaces desktop

## ⚙️ Funcionalidades do Sistema

O sistema possui as seguintes funcionalidades:

### Usuário

- Cadastro de conta
- Login no sistema
- Realizar agendamentos
- Visualizar agendamentos

### Administrador

- Visualizar histórico de agendamentos
- Editar agendamentos
- Excluir agendamentos
- Gerenciar serviços

---

## 🗄️ Estrutura do Banco de Dados

O banco de dados foi projetado para gerenciar:

- Usuários
- Agendamentos
- Histórico de ações administrativas

Relacionamentos principais:

- Usuário → pode possuir vários agendamentos (1:N)
- Agendamento → possui histórico de ações
- Administrador → pode gerenciar agendamentos
